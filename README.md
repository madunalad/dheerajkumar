# Dheeraj Kumar - Portfolio

Personal portfolio site built with pure HTML, CSS, and JavaScript. No frameworks, no build tools, no dependencies.

## Deploy on GitHub Pages

1. Go to your repo `madunalad.github.io`
2. Click **Add file - Upload files**
3. Upload `index.html`
4. Click **Commit changes**
5. Go to **Settings - Pages - Source: main branch - Save**

Live at `https://madunalad.github.io` within 60 seconds.
